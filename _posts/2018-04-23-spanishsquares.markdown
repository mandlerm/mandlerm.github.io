[Click here to see a walkthrough](https://youtu.be/O3sPgbYnhmw)


[GitHub Repo](https://github.com/mandlerm/SpanishBingo/tree/min_game GitHub Repo)

 Tech Stack: Postgres, Rails, React, Redux


Created a Rails API with a PostgreSQL database
Utilized React frontend with a Redux store using ES6 where possible
Chose CSS Grid for page layout

I am in the process of redoing this app without Redux (it is not needed for the demands of the project).
I'll be focusing more on design, enabling user accounts, and allowing users to use 'stock' cards or create their own.
